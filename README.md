## <a name="getting"></a> Getting Started:

1. Install dependencies

   ```bash
   npm i
   ```

2. Fire up the server and watch files

   ```bash
   npm start
   ```

## <a name="deployment"></a> Deployment:

1. Builds the app for production to the build folder.

   ```bash
   npm run build
   ```
